# 1. コンセプト
**「目覚めを待つ王国」**
生徒の意気込みが光となり森に宿る。観客の感想が祝福となり城を照らす。本番当日、王国が目覚める。発表会そのものを物語化する装置としてのウェブアプリ。

## 優先する軸
**「3. 世界観演出を極限まで高める」** と **「1. 生徒の意気込みを主役にする」**
美しいUIとインタラクションによって、バレエの世界観に入り込んだような没入感を提供し、生徒のモチベーションと観客の期待感を最大化します。

# 2. 全体構造
- **公開エリア（観客/一般）**: 世界観トップページ、カウントダウン、公演情報
- **生徒エリア（Phase 1ではモック）**: 意気込み投稿
- **運営エリア（Phase 3以降）**: 管理ダッシュボード

# 3. デザインシステム（UI/UX）
- **テーマカラー**:
  - Primary: ロイヤルブルー（#0B192C, #1E3E62）
  - Accent: ゴールド（#D4AF37, #FBF8CC）
  - Text: クリームホワイト（#F8F9FA）
- **タイポグラフィ**: 
  - 見出し: 明朝体（Shippori Mincho, または Noto Serif JP）
  - 本文: モダンゴシック（Inter, Noto Sans JP）
- **演出（アニメーションと奥行き）**:
  - Glassmorphism（ガラスのような透け感）
  - Particle（意気込みや魔法を表現する光の粒のアニメーション）
  - Parallax（視差効果）による3層構造の立体感：
    1. **遠景（背景）**: うっそうとした深い森
    2. **中景**: 眠りについた城、もしくはタイトルタイポグラフィ
    3. **近景（前景）**: オーロラ姫や群舞のシルエットを大きく配置し、視線を中央や上部へ誘導する。スクロールに合わせて手前のシルエットが異なる速度で動く立体空間を生成。
